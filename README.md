## 🚀 Uso

Solo abre `index.html` en tu navegador. Cada animación está en su propia tarjeta con el nombre y la técnica usada.

Para copiar el código de una animación:
1. Abre `index.html`
2. Busca la animación que quieres en el `<style>`
3. Cópiala a tu proyecto

## 🛠️ Tecnologías

- HTML5 semántico
- CSS puro (sin frameworks)
- Vanilla JS (solo para el contador y filtros)
- Google Fonts (Space Mono + Syne)

## 📁 Estructura


## 🤝 Contribuir

¿Quieres agregar más animaciones? ¡Pull requests bienvenidos!

1. Fork el repo
2. Agrega tu animación en el `<style>` con un comentario claro
3. Agrega la tarjeta correspondiente en el HTML
4. Abre un Pull Request
